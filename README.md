A cinematic, scroll-driven travel experience showcasing five iconic Colombian cities.

VoyageVista is a high-end, full-screen travel catalog site built with **React**, **Vite**, and **TailwindCSS**.  
Each section uses vertical **scroll-snap**, cinematic background visuals, smooth transitions, and elegant typography to create an immersive travel experience.

Originally developed on Replit using AI-assisted workflows, VoyageVista is now fully migrated to GitHub for continued development with Code Assistants and local tooling.

- **Full-screen scroll-snap layout**  
  Every section is 100vh for a clean, presentation-style flow.

- **Cinematic background videos**  
  Each featured Colombian city has its own unique AI-generated video backdrop.

- **Smooth transitions & animations**  
  Subtle fade-ins and scroll effects create a polished, cinematic feel.

- **TailwindCSS styling**  
  Responsive, modern UI using utility-first design.

- **React + Vite architecture**  
  Extremely fast hot reloading and optimized production builds.

- **Easy deployment**  
  Works seamlessly with Vercel, Netlify, GitHub Pages, or any static host.
